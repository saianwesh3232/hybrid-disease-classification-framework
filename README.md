# Multi-Class Disease Detection System

This project is a deep learning-based disease detection system developed using CNN and VGG16 architecture. It classifies medical images into different disease categories and includes a Flask-based web interface for prediction.

## Technologies Used
- Python
- TensorFlow
- Keras
- Flask
- HTML
- CSS
- JavaScript

## Features
- Image classification using CNN and VGG16
- Medical image preprocessing
- Model evaluation using confusion matrix and ROC-AUC
- Flask-based web deployment
- Real-time image prediction

## Project Structure
- app.py: Flask backend
- templates/index.html: Frontend page
- model/: Trained model file
- notebook/: Training notebook
- static/uploads/: Uploaded images

## Important Note
The trained .h5 model file is not included in this repository due to GitHub file size limitations.
